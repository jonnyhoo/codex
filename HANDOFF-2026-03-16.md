## Completed changes

### 1. Instruction layers are now typed and runtime-visible

- Added typed instruction sections with:
  - `audience`
  - `priority`
  - `source`
- `project_doc` now resolves ordered contextual-user instruction sections and then joins them back into the existing wrapped user-instructions format to avoid prompt-format churn.
- `Session::resolve_instruction_layers()` assembles a single resolved instruction view before rendering prompt items.
- `RuntimeContext.instructions` now exposes:
  - `user_instruction_sections`
  - `resolved_layers` for tool-side runtime inspection

### 2. RuntimeContext now carries agent-scoped collaboration metadata

- Added `RuntimeAgentContext` and attached it to `RuntimeContext`.
- The runtime snapshot now includes:
  - `agent_id`
  - `parent_session_id`
  - `depth`
  - `agent_nickname`
  - `agent_role`
  - structured `subagents`
- `ToolInvocation::runtime_context()` now asynchronously enriches the snapshot with:
  - resolved instruction layers
  - structured child-agent status entries
